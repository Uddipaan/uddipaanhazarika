Experiences

I have worked on many awesome projects and technologies. Being a dev enthusiast, I am just obsessed with new advancement in the fields of Computer Science. Wanna chat with me regarding any projects or ideas? Just hit me up on my social media.

QSYS

Its an awesome ticketing system used by business and field users to raise and resolve queries. The system deals a daily hit of more than thousand concurrent users. Qsys is written solely in Java using Spring MVC as its web framework. It consists of complex workflow mechanisms which facilitates queries with accordance to different business groups.

I-Smart

What is more trendy than artificial intelligence in our modern computing era? I-Smart is a facial recognition system for HR platforms used during enrolling new resources. It is a face verification system used to verify the exactness of a person's face in a document, to his actual face. This is written solely in python and uses HOG in its underlying mechanisms.

Text Summarizer

How nice it would be if a system could automatically generate a summary of a long prose. Text Summarizer is an restful API based system written both in Django as well as Flask(coz why not!). It takes in long data string and summarizes it, and returns a json string containing the summary of the input string. Internally it uses features provided by nltk and spacy.

CI-CD

CI-CD is a jargon term that we hear nowadays in the IT industry. But it is just a philosophy of doing things being independent of operations as I see it. In one of my recent venture of experiencing so call CI-CD and attaining "Agile-enlightment", I built a continous pipeline with components: Version Control, Build System, Docker, Asset Repository, Kubernetes and finally an CI tool.

Scrapping and Clustering

One of my ongoing venture is about scrapping data and clustering them with respect to certain parameters. The domain I am interested in is the Zomato restaurants dataset that can be scrapped easily and can be visualised and different operations can be done on them to find meaningful conclusions. Technologies used is Python and libraries such as pandas etc with beautiful soap/scrappy and also selenium web driver.

SAT Solvers

In studies of satisfiability problem, the problem of counting distinct solutions for any boolean formula is called Model Counting. Here, I implemented two of the leading state of the art model counting algorithms in python. It was an analytical study of the algorithms.
